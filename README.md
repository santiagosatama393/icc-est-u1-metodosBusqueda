## Praxtica 5 :busqueda binaria 
## Fecha:2026/18/05
### Descripcion:
Este proyecto implementa el algoritmo de búsqueda binaria en Java utilizando un arreglo de objetos Product. Primero se ordenan los productos por nombre con el método burbuja y luego se realiza la búsqueda binaria para encontrar un producto específico de manera rápida y eficiente. El programa muestra la posición donde se encuentra el producto o un mensaje indicando que no fue encontrado.
